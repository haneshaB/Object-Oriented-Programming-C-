# CS2810 – Object-Oriented Programming & Data Structures Lab Projects

A collection of course projects developed as part of **CS2810: Object-Oriented Programming and Data Structures (C++)**. This repository demonstrates the application of Object-Oriented Design, Generic Programming, Dynamic Memory Management, Exception Handling, and advanced Data Structures & Algorithms through hands-on assignments and mini-projects.

---

## 🚀 Tech Stack

- **Language:** C++17
- **Paradigms:** Object-Oriented Programming, Generic Programming
- **Libraries:** STL, File I/O
- **Core Concepts:** Templates, Inheritance, Polymorphism, Operator Overloading, Exception Handling, Dynamic Programming

---

## 📂 Projects

### 🔹 Lab 1 – Matrix Operations
Implemented a dynamic `Matrix` class supporting:

- Dynamic 2D memory allocation
- Trace computation
- Upper/Lower triangular matrix detection
- In-place 90° matrix rotation
- Operator overloading for matrix arithmetic
- Runtime polymorphism using inheritance

**Concepts:** Dynamic Memory Allocation, Operator Overloading, Inheritance, Polymorphism.

---

### 🔹 Lab 2 – Bookshelf Management System
A linked-list-based bookshelf management application featuring:

- Vertical and Horizontal bookshelves
- Book insertion and deletion
- Reversing linked lists
- Group-wise reversal (`reverseK`)
- Abstract base classes and function overriding

**Concepts:** Linked Lists, Inheritance, Function Overriding, Abstract Classes.

---

### 🔹 Lab 3 – MagicalCrate: Dynamic Storage System
A template-based container with automatic resizing policies.

Features:

- Generic dynamic container implementation
- Automatic expansion and contraction
- Global template functions:
  - Linear Search
  - Binary Search
  - Sorting
- Largest element removal
- Manual memory management without STL containers

**Concepts:** Templates, Dynamic Arrays, Generic Programming, Memory Management.

---

### 🔹 Lab 4 – Robotic Warehouse Controller
A warehouse inventory management system with transactional guarantees.

Features:

- Generic stack implementation
- Custom exception hierarchy
- Atomic bulk operations with rollback support
- Weight and energy constraints
- Inventory management

**Concepts:** Templates, Exception Handling, Stack ADT, Transaction Rollback, Strong Exception Safety.

---

### 🔹 Lab 5 – High-Frequency Trading Utility
A mini trading analytics and execution engine.

Features:

- CSV parsing and file processing
- Market order analytics
- Trader exposure analysis
- Transaction logging
- Order matching engine

**Concepts:** File I/O, STL Containers, CSV Processing, Financial Data Processing.

---

### 🔹 Lab 6 – Intelligent Spell Checker
A rule-based spell correction engine supporting:

1. Exact Match
2. Case-Insensitive Match
3. Anagram Match
4. Vowel Error Match

**Concepts:** Hashing, String Manipulation, Search Optimization.

---

### 🔹 Lab 7 – Meeting Room Scheduler
A greedy scheduling system for efficient room allocation.

Features:

- Optimal room assignment
- Conflict resolution
- Active meeting queries
- Most-used room detection
- Longest meeting analysis

**Concepts:** Greedy Algorithms, Sorting, Priority Queues, Interval Scheduling.

---

### 🔹 Lab 8 – Deterministic Skip List
A template-based deterministic skip list implementation.

Features:

- Insert
- Delete
- Search
- Level-wise traversal and printing
- Deterministic level construction

**Concepts:** Skip Lists, Templates, Multi-Level Linked Lists.

---

### 🔹 Lab 9 – Dynamic Programming Toolkit
Implementation of classical DP problems:

- 0/1 Knapsack
- House Robber
- Partition into Two Sets
- Solution reconstruction via backtracking

**Concepts:** Dynamic Programming, State Transition Design, Backtracking.

---

### 🔹 Lab 10 – CSV Analyzer & Generic Data Processing
A type-aware CSV processing and analytics system.

Features:

- Templated column representation
- Statistical computations:
  - Minimum
  - Maximum
  - Mean
  - Variance
- Custom exception hierarchy
- Runtime polymorphism

**Concepts:** Templates, Polymorphism, Exception Handling, Data Analytics.

---

### 🔹 Mid-Sem Utility Toolkit
A collection of algorithmic utilities including:

- Generic threshold filtering
- Polymorphic billing calculator
- Valley detection algorithms
- Particle collision simulation

**Concepts:** Templates, Polymorphism, Stack Simulation, Array Algorithms.

---
